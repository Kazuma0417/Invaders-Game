# Invaders Game

シンプルなPygame製のインベーダーゲームです。

## 遊び方

* 左右矢印キーでプレイヤーを移動します。
* スペースキーで弾を発射します。

## 実行方法

1.  Python 3.x がインストールされていることを確認してください。
2.  Pygameライブラリをインストールします。
    ```bash
    pip install pygame
    ```
3.  このリポジトリをクローンまたはダウンロードします。
    ```bash
    git clone [https://github.com/YourUsername/YourGameRepo.git](https://github.com/YourUsername/YourGameRepo.git)
    cd YourGameRepo
    ```
4.  ゲームを実行します。
    ```bash
    python sample.py
    ```

## 必要なファイル

* `sample.py`
* `player.png`
* `enemy.png`
* `bullet.png`
